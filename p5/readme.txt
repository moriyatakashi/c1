Bad Apple!! PV-FC 2.5 簡易マニュアル

●操作方法
＊通常時
・左     : 1/2 倍速再生
・A      : 先頭から再生
・B      : [再生時] 停止
　         [停止時] 色とドットパターンの初期化
・START  : メニューに入る
・SELECT : ドットパターンの変更 (3 種類)

＊メニュー時
・上下   : カーソルの移動
・左右   : 色の変更
・B      : 色の初期化
・START  : メニューから出る
・SELECT : ドットパターンの変更 (3 種類)


●メニュー項目
＊SHADOW : 影部分の色
＊LIGHT  : 光部分の色
＊IMGLAG : 残像部分の色


●備考
＊使用したサウンドドライバに一時停止機能が存在しないため、
　各チャンネルを無理矢理有効/無効にすることで
　曲の一時停止を実現しています。そのため一時停止解除時に、
　一部の音がすぐに鳴らない場合がありますが仕様です。

＊IMGLAG 指定により、偶数コマ表示時に、
　直前の奇数コマでのみ表示されていた影部分に色をつけることで、
　疑似的に残像を表現します (30 fps の実現に伴う副作用です)。
　残像部分は偶数コマでのみ表示されるため、
　色の組み合わせによっては激しい点滅として見える場合があります。
　ご注意下さい。

＊IMGLAG は LIGHTの変更に追従します
　(LIGHT と IMGLAGを同色にすることで残像表示が無効になります)。

＊メニューでの十字キー入力はリピートします。


●ROM情報
